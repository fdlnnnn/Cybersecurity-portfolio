# Project Experience
**Project 1**

Penilaian Kerentanan Pada Aplikasi Web E-Commerce.

* Tujuan: Menemukan celah keamanan sistem pada aplikasi web, untuk mengidentifikasi dan memitigasi resiko keamanan.
* Tools: Nmap, dirb, burp suite.

# Metodologi:
1. Reconnaissance:
   * namp: Digunakan untuk mendeteksi OS, versi service, nmap scripting engine, dan melihat jalur paket data pada target.
   * dirb: untuk mencari directory" yang ada pada aplikasi target.
2. Exploitation & Post Explooitation:
   * Memanfaatkan informasi sensitif dari hasil reconnaissance untuk masuk kedalam sistem.
   * Melakukan hal-hal setelah melakukan eksploitasi seperti, mapping aplikasi, dan menilai dampak dari kerentanan yang ditemukan. Tools yang digunakan yaitu: burp suite dan SQLmap.
# Output
* Berhasil menemukan 7 kerentanan yaitu: Information Disclosure (Critical), SQLinjection (Critical), IDOR (High), XSS (High), Unrestricted file upload (high), CSRF (high), Credential Stuffing (Medium).
* Membuat laporan confidential hasil penetration testing yang lengkap dengan mitigasi dan rekomendasi tindakan lebih lanjut.
